# cgame

## Motivation

Someone wanted to c me write in C. So here's a puzzle game written in C99.

## Installation: Ubuntu

This app requires some packages installed:

    apt-get install libsdl2-dev libsdl2-image-dev

## Usage

Compile and run:

    ./start.sh

Run unit tests:

    ./start.sh test
